                      Задание

1. Опишите все известные вам способы позиционирования 5
 элементов по центру, каждый из которых должен занимать 20%.

2. Опишите все известные вам способы позиционирования
          модального окна, по центру экрана.

3. Напишите стили для кнопки, которая может иметь несколько
  состояний: active, disabled, danger. Цвета произвольные.

4. Сверстайте страницу логина, которая должна
содержать только форму с полями email, password.
